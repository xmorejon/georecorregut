# GeoRecorregut

Is an app to keep track the places you have visited and highlight how many countries you have travelled to.

It has user management, GoogleMaps API integration, Multilanguage feature, and some statistics to compare yourself with other registered users.

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

Following keys are needed and placed in .env file (or Secrets):
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY="**KEY**"
NEXT_PUBLIC_GOOGLE_MAPS_MAP_ID="**KEY**"
GEMINI_API_KEY="**KEY**"
GOOGLE_API_KEY="**KEY**"
