# **App Name**: GeoRecorregut

## Core Features:

- Interactive Map: Visually display visited locations on a world map.
- Automatic Location Recording: Ability to record location coordinates with time/date automatically.
- Statistics: Statistics about how much world I have visited, displaying metrics and percentages on number of continents, countries: Statistics about how much world I have visited, displaying metrics and percentages on number of continents, countries.
- Location Search: Search bar to find and display past locations.
- Data Import: Import locations via CSV files, GPX files or Google Location History.
- User Authentication: Provide secure user authentication to prevent unauthorized access.
- Data Export: Allow users to export their travel data.
- Multilanguage setting: Application default language should be Catalan, but it should also be possible to select Spanish and English

## Style Guidelines:

- Primary color: A warm blue (#5DADE2) evocative of travel and open skies.
- Background color: Very light blue-gray (#F0F4F7), almost white, to suggest cartography and history, keeping the focus on the map.
- Accent color: Muted teal (#45B39D) for interactive elements and points of interest.
- Body and headline font: 'PT Sans' (sans-serif), modern and clean, ensuring readability.
- Simple and clean icons for map markers, search, and navigation.
- Map should take central space, with side panels/drawers for location search and settings.
- Smooth transitions and subtle map animations when displaying and moving between locations.
